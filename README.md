# Digitalisasi Pemesanan Makanan di Rumah Makan Padang 

# Permasalahan 
Rumah makan Padang dikenal dengan sistem penyajian cepat, tetapi pada jam sibuk sering 
terjadi antrian panjang, baik untuk: 
• memesan makanan di tempat, 
• menunggu meja kosong, 
• maupun untuk take-away. 

Masalah lain yang sering muncul: 
• Konsumen tidak tahu kondisi tempat makan, apakah penuh atau masih ada meja 
kosong. 
• Antrian membuang waktu, terutama bagi pelanggan yang hanya punya waktu makan 
singkat. 
• Kesalahan pemesanan (karena pesanan disampaikan manual ke kasir). 

# Tujuan Pengembangan Aplikasi 
1. Mengurangi antrian fisik di rumah makan Padang melalui sistem pemesanan digital. 
2. Mempermudah pelanggan untuk: 
o Memesan tempat (meja), 
o Melihat ketersediaan tempat, 
o Memesan makanan/minuman langsung dari aplikasi. 
3. Meningkatkan efisiensi operasional bagi pemilik rumah makan melalui otomatisasi 
sistem pemesanan. 
4. Meningkatkan kenyamanan pelanggan baik untuk makan di tempat maupun pesan 
antar.

# Penyelesaian (Konsep Aplikasi) 
Aplikasi ini akan memiliki dua sisi pengguna: 
# 1. Untuk Pelanggan 
• Fitur utama: 
1. Melihat daftar rumah makan Padang yang terdaftar. 
2. Mengecek ketersediaan meja secara real-time. 
3. Melakukan reservasi meja untuk makan di tempat. 
4. Memesan makanan dan minuman langsung dari aplikasi. 
5. Pilihan “Makan di Tempat” atau “Pesan Antar”. 
6. Pembayaran melalui QRIS, e-wallet, atau tunai di tempat. 
7. Mendapatkan notifikasi status pesanan (diproses, siap, diantar, selesai).
   
# 2. Untuk Pemilik Rumah Makan 
• Fitur utama: 
1. Menerima dan mengelola pesanan masuk (dari meja atau online). 
2. Memantau status meja (kosong, dipesan, terisi). 
3. Mengatur menu dan harga di aplikasi. 
4. Melihat laporan harian penjualan dan statistik pesanan.
   
# Alur Kerja Aplikasi 
1. Pelanggan membuka aplikasi → login/register. 
2. Melihat status ketersediaan meja. 
3. Memilih meja → memesan makanan/minuman dari menu digital. 
4. Pesanan otomatis dikirim ke dapur dan kasir. 
5. Setelah makanan siap → diantarkan ke meja sesuai nomor pesanan. 
6. Jika pelanggan memilih delivery, kurir rumah makan akan mengantar pesanan.
   
# Alur Kerja Sederhana 
1. Pelanggan membuka aplikasi → login/register. 
2. Melihat status ketersediaan meja. 
3. Memilih meja → memesan makanan/minuman dari menu digital. 
4. Pesanan otomatis dikirim ke dapur dan kasir. 
5. Setelah makanan siap → diantarkan ke meja sesuai nomor pesanan. 
6. Jika pelanggan memilih delivery, kurir rumah makan akan mengantar pesanan.
   
# Tujuan Akhir 
Menghadirkan solusi digital yang: 
• Ramah pengguna (user-friendly), 
• Menghemat waktu pelanggan, 
• Meningkatkan efisiensi rumah makan, 
• Mendukung transformasi digital usaha kuliner lokal seperti Rumah Makan Padang.
